Census Demo Layers
================

## Libraries

``` r
library(tidycensus)
library(sf)
```

    ## Linking to GEOS 3.7.1, GDAL 2.3.2, PROJ 5.2.0

## Fetch layers

``` r
for (istate in c("GA", "NV")) {
  for (itable in c("B28002", "B28010")) {
    assign(
      paste0(istate, "_", itable),
      tidycensus::get_acs(
        geography = "tract",
        table = itable,
        year = 2017,
        output = "wide",
        state = istate,
        geometry = TRUE,
        moe_level = 90,
        survey = "acs5"
)
    )
  }
}
```
